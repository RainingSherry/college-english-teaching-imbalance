# 论文主要文件说明

## 目录结构

- `01_代码/分析.py`：生成主论文结果的核心分析脚本。
- `02_数据/college_english_teaching_evaluation.csv`：主实验使用的数据集。
- `03_结果/`：主实验已经生成的报告、表格和图表结果。

## 复现实验

可从任意位置运行：

```bash
python 主要文件夹/01_代码/分析.py
```

脚本会自动读取 `主要文件夹/02_数据/college_english_teaching_evaluation.csv`，并将新结果输出到 `主要文件夹/03_结果/`。

## 收纳目录

- `../其他数据收纳/二次实验/`：二次实验代码、数据副本和结果。
- `../其他数据收纳/Python缓存/`：语法检查时生成的 Python 缓存。
