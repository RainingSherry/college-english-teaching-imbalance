# 大学英语教学质量评估：类别不平衡与机器学习

本仓库整理了论文实验相关的主要代码、数据和结果。

## 主要目录

- `主要文件夹/01_代码/分析.py`：主论文结果复现脚本。
- `主要文件夹/02_数据/college_english_teaching_evaluation.csv`：主实验数据。
- `主要文件夹/03_结果/`：主实验输出结果，包括报告、表格和图像。
- `其他数据收纳/二次实验/`：二次实验代码、数据副本和结果。

## 运行主实验

```bash
python 主要文件夹/01_代码/分析.py
```

脚本会自动读取 `主要文件夹/02_数据/college_english_teaching_evaluation.csv`，并把结果输出到 `主要文件夹/03_结果/`。

## 依赖

建议使用 Python 3.10 或 3.11 创建独立环境，然后安装：

```bash
pip install -r requirements.txt
```
